Sistema operativo:
Es el software principal que administra los recursos del computador, como la memoria, los archivos y los programas. Permite que las aplicaciones funcionen correctamente.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones del lado del servidor.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas, las respuestas y el comportamiento de la aplicación cuando recibe solicitudes.

Puerto:
Es un punto de comunicación que permite a los programas enviar y recibir datos. En este caso, el servidor escucha en el puerto 3000 para atender peticiones.

Logs:
Son registros de eventos y actividades que ocurren en el sistema o la aplicación. Ayudan a monitorear el funcionamiento y a identificar errores o problemas.
